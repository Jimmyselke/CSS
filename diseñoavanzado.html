Explicación de CSS Margin, Padding, Border, Width y Height

1. CSS Margin

El margin es el espacio que rodea un elemento, pero que se encuentra fuera de su borde. Es útil para separar elementos entre sí. En este ejemplo, el margen es el espacio entre el borde color tomate y el borde de la página o cualquier otro elemento cercano.

2. CSS Padding

El padding es el espacio interior de un elemento, entre su contenido y su borde. En este ejemplo, el padding está representado por el espacio azul acero entre el borde morado y el contenido dorado.

3. CSS Border

El border es la línea que rodea el padding y el contenido. Puedes ajustar el grosor, color y estilo del borde. En este ejemplo, el borde exterior es de color tomate y el borde interior es morado, separando los diferentes niveles de contenido.

4. CSS Width y Height

Las propiedades width (ancho) y height (altura) controlan el tamaño de los elementos. En este ejemplo, el contenedor tiene un ancho de 500px y una altura de 300px, ajustados por los valores del padding y border.
