Explicación de Margin, Padding, Border, Width y Height

1. CSS Margin

El margin en CSS se utiliza para crear espacio alrededor de los elementos, fuera de sus bordes. En este ejemplo, el margen está representado por el espacio entre el borde verde oscuro y el borde de la página.

2. CSS Padding

El padding en CSS se refiere al espacio interior de un elemento, es decir, el espacio entre el contenido y el borde del elemento. En este ejemplo, el relleno es el área blanca entre el contenido naranja y el borde azul oscuro.

3. CSS Border

El border en CSS es el borde que rodea el padding y el contenido. Puedes especificar el grosor, el estilo y el color del borde. En este ejemplo, hay dos bordes: uno alrededor del contenedor principal (verde oscuro) y otro alrededor del contenido (azul oscuro).

4. CSS Width y Height

Las propiedades width (ancho) y height (altura) en CSS se utilizan para definir el tamaño de un elemento. En este ejemplo, el ancho del contenedor es de 500px y la altura es de 300px. El contenido interno tiene un relleno adicional que también ajusta su tamaño.
