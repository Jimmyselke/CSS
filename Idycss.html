No repetir el id: El atributo id debe ser único en la página. Si se repite, causará problemas con CSS y JavaScript, ya que estos solo reconocerán el primer elemento con ese id.

Especificidad id vs clase: En CSS, los id tienen más peso que las clases. Si un elemento tiene ambos, el estilo definido por el id prevalecerá sobre el de la clase.
